<task type="auto">
  <name>Task 1: Create contract deployment script</name>
  <files>contracts/deploy.ts</files>
  <action>
    Create deployment script using 'one' CLI:

    ```typescript
    import { execSync } from 'child_process';
    import * as fs from 'fs';
    import * as path from 'path';

    export async function deployContract() {
      // 1. Compile Move package
      console.log('Compiling Move package...');
      execSync('one move compile', { cwd: __dirname, stdio: 'inherit' });

      // 2. Deploy to testnet
      console.log('Deploying to testnet...');
      const output = execSync('one move publish --network testnet', {
        cwd: __dirname,
        encoding: 'utf-8',
      });

      // 3. Parse deployed address from output
      const addressMatch = output.match(/0x[a-fA-F0-9]+)/);
      const deployedAddress = addressMatch ? addressMatch[0] : null;

      if (!deployedAddress) {
        throw new Error('Could not parse deployed address');
      }

      // 4. Update Move.toml with deployed address
      const tomlPath = path.join(__dirname, 'Move.toml');
      let tomlContent = fs.readFileSync(tomlPath, 'utf-8');
      tomlContent = tomlContent.replace(
        /minesweeper_bet = "0xYOUR_DEPLOYED_ADDRESS_HERE"/,
        `minesweeper_bet = "${deployedAddress}"`
      );
      fs.writeFileSync(tomlPath, tomlContent);

      console.log(`Deployed to: ${deployedAddress}`);
      return deployedAddress;
    }
    ```

    The script should output the deployed address for manual verification on OneScan.
